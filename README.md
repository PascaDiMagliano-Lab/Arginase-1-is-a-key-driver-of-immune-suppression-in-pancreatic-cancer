# Arginase-1-is-a-key-driver-of-immune-suppression-in-pancreatic-cancer
This repository includes the code used for data processing and visualization of single cell RNA sequencing data as presented in the manuscript **“Arginase 1 is a key driver of immune suppression in pancreatic cancer”.** Running this R script as uploaded will generate Figures  2G-J, 2S2B-E, 2S3C, 2S3F-H, 3B-F, and 3I found in the manuscript. Raw data files are available through the NIH Gene Expression Omnibus (GEO), accession numbers [GSM6149735](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE203016) (11 month KF) and [GSM6149736](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSM6149736) (11 month KFCA).
